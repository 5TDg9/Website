## 功能

* 支持 免ROOT(DPM)、ROOT(HIDE)、ROOT(DISABLE)、系统应用(DISABLE)、系统应用(DISABLE\_USER)、系统应用(DISABLE\_UNTIL\_USED) 等模式
* 支持计划任务
* 支持定时任务
* 支持条件任务
* 支持延时任务
* 支持一键冻结
* 支持一键解冻
* 支持离开冻结
* 支持分类查看
* 支持通知栏瓷块一键冻结解冻锁屏
* 支持锁屏后自动冻结
* 支持多种快捷方式操作
* 支持多种解冻/冻结方式
* 支持更换界面风格
* 支持备份与恢复
* 等更多功能！
