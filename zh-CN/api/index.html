# API

自冻FreezeYou提供了一些接口供外部调用，具体情况如下：

  • 通过 URI 唤起 自冻(FreezeYou) 冻结解冻应用程序
  • 通过 Provider 唤起 自冻(FreezeYou) 进行冻结解冻以及数据查询等操作
  • 通过 startActivity 唤起 自冻(FreezeYou) 冻结解冻应用程序

后续如新增一些对外接口以及接口能力，此页也会进行更新。
